# Orlov Logistics — лендинг логистической компании

Одностраничный сайт компании, занимающейся поставками и доставкой грузов из Китая: услуги, способы доставки, этапы работы и форма заявки с отправкой лида в Telegram.

## Возможности

- Адаптивный одностраничный лендинг с мобильным меню-«шторкой»
- Блоки услуг: карго и официальная доставка, поиск фабрики/поставщика, инспекция товара, платёжный агент, OEM/ODM-брендирование
- Описание способов доставки (авиа, авто, ж/д, море) и видов упаковки
- Секции «Этапы работы», «С кем мы работаем», команда, отзывы и FAQ
- Промо-слайдер и анимации появления секций при скролле (IntersectionObserver)
- Формы заявки (полная и мини-форма): отправка данных в Telegram через серверный PHP-прокси `send.php`
- Cookie-баннер и отдельные страницы политики, приватности и cookies
- Конфигурация секретов вынесена в `.example`-файлы (токен и chat ID не хранятся в репозитории)

## Стек

- HTML5, CSS3 (кастомные стили, glassmorphism), шрифт Inter
- Ванильный JavaScript (без фреймворков)
- PHP — серверный прокси для отправки сообщений в Telegram Bot API (`send.php`)

## Структура

```
index.html               # главная страница лендинга
styles.css               # стили
script.js                # логика форм, слайдеров, анимаций
send.php                 # серверный прокси для отправки заявок в Telegram
env.example.js           # пример клиентской конфигурации (TG_CONFIG)
env.server.example.php   # пример серверной конфигурации (токен, chat ID)
cookies.html             # страница про cookies
policy.html              # пользовательское соглашение
privacy.html             # политика конфиденциальности
assets/                  # иконки (SVG)
images/                  # изображения и логотип
```

## Запуск

Статический сайт — для просмотра достаточно открыть `index.html` или поднять локальный сервер:

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Для работы формы заявки нужен PHP (отправка в Telegram через `send.php`):

```bash
php -S localhost:8000
```

Настройка отправки лидов:

1. Скопируйте `env.server.example.php` в `env.server.php` и укажите свои `TELEGRAM_TOKEN` и `TELEGRAM_CHAT_ID`.
2. При необходимости скопируйте `env.example.js` в `env.js` и задайте `TG_CONFIG`.

Файлы с реальными секретами не коммитятся (исключены через `.gitignore`).
